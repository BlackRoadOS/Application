## Tech Stack

- **Edge Compute**: Cloudflare Workers + D1 (SQLite)
- **AI Inference**: Workers AI (Llama 3.1) + Ollama on Pi fleet
- **Hardware**: 5x Raspberry Pi 5 + 2x Hailo-8 NPU (52 TOPS)
- **Blockchain**: PS-SHA∞ hash chains + RoadChain D1 ledger
- **Frontend**: Pure JS, no framework, brand.css design system
- **Auth**: Clerk + CarKeys
- **Payments**: Stripe + Coinbase Commerce + RoadCoin
- **DNS**: Cloudflare + self-hosted PowerDNS
- **VPN**: WireGuard mesh + Tailscale
