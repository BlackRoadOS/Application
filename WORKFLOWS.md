# BlackRoad OS — Cross-Product Workflows
> Everything flows through CarPool (shared memory + triggers) and RoadChain (provenance on every action).
> You stay in one browser tab. Agents hand off naturally. RoadCoin rewards the entire ride.

---

## 1. BlackRoad OS — The Highway
**Start your day** → CarKeys authenticates → RoadSide checks setup → you're driving.
All 14 products launch from the dock or Cmd+K. Persistent state syncs across devices.

## 2. RoadCode — Build & Ship
Code → Claude co-pilot → test in-browser → one-click deploy.
Deploy triggers RoadWork (invoice, announce). Commits stamped on RoadChain. BackRoad posts the launch.

## 3. CarPool — AI Integration Hub
Import external AI conversations → they join shared memory → set hand-off triggers.
Claude research → CarPool hands to RoadTrip Eve → verified by RoadView → sent to BlackBoard for visuals.

## 4. OneWay — Data Ownership
Set destination once → nightly delta exports → full manifest with RoadChain proof.
Exports include Shared Memory, BlackBoard assets, RoadCode code, RoadWork logs.

## 5. RoadSide — Onboarding
Conversational setup → auto-imports data → creates CarKeys → pre-loads agents → generates welcome assets.
Hands off to CarPool for AI imports and RoadView for search tuning.

## 6. RoadView — Verified Search
Ask anything → searches your data first, then web + blockchain → answer with proof badges.
"Ask the Convoy" button hands queries to RoadTrip agents. Results feed BlackBoard and RoadWork.

## 7. RoadTrip — Agent Convoy
Group chat with 18 agents → shared memory → natural hand-offs.
Eve researches → Pixel designs → Cadence writes → BackRoad posts → RoadWork tracks results.

## 8. BackRoad — Social Autopilot
Content from any product → auto-formatted and scheduled → agents engage on your behalf.
Performance data feeds back to RoadView and RoadCoin earnings.

## 9. RoadWork — Business Automation
Set goals in Steer Mode → 7 agents execute (invoicing, support, marketing, ops).
New sale → auto-invoice → support follow-up → marketing promotion. Crew huddles summarize weekly.

## 10. Roadie — Education Companion
Student starts session → Socratic guidance → progress tracked → portfolio built.
RoadView verifies facts → BlackBoard creates visual aids → portfolio saved on RoadChain.

## 11. BlackBoard — Creative Studio
Prompt or import idea → generate assets → optimize for platforms → publish.
RoadTrip agents collaborate → RoadChain stamps provenance → BackRoad distributes.

## 12. CarKeys — Security Layer
Biometric login → auto-rotates credentials → monitors devices.
Required for every product launch. Scoped keys for agent access. All logged on RoadChain.

## 13. RoadChain — Verification Foundation
Every action automatically hashed and anchored.
Powers proof in RoadView, provenance in BlackBoard, audit in RoadWork, integrity in OneWay.

## 14. RoadCoin — Reward Engine
Earn on actions across all products → spend in Marketplace → stake for governance.
Complete a RoadTrip → earn → spend on BlackBoard boost → create better → post → earn more.

Morning: CarKeys authenticates → RoadSide checks setup → RoadView searches overnight updates → RoadTrip convoy brainstorms → BlackBoard creates assets → BackRoad posts on autopilot → RoadWork handles customers and invoicing → RoadChain stamps everything → RoadCoin balance updates.

**You never leave the single browser tab. Memory flows. Agents collaborate. Everything is verified. You earn while you drive.**
